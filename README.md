# GloryProtect

GloryProtect — survival roguelike с видом сбоку, в котором игрок управляет магически парящей платформой, поддерживает пять секций глобального защитного щита и защищает экипаж от непрерывного абордажа.

## Основная идея

Платформа перемещается только по горизонтали и постоянно находится под воздействием ветра. Чтобы восстановить повреждённую секцию щита, игрок должен добраться до связанного с ней наземного энергетического шара, совместить его с шаром платформы и удерживать контакт.

Якоря позволяют стабилизировать платформу, но каждый установленный якорь одновременно становится путём, по которому враги могут подняться на борт. Игрок распределяет ограниченный экипаж между управлением, якорями, боем, лечением и турелями.

## Жанр и формат

- Survival roguelike.
- Платформер с видом сбоку.
- Один бесконечный по сложности забег.
- Основной результат: время выживания и количество убийств.
- Управление платформой через физическое горизонтальное усилие, ветер и инерцию.
- Пять независимых секций щита.
- Две независимые угрозы: стратегические волны и физический абордаж.
