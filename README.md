# GloryProtect

GloryProtect — survival roguelike с видом сбоку, в котором игрок управляет магически парящей платформой, поддерживает пять секций глобального защитного щита и защищает экипаж от непрерывного абордажа.

## Движок

Проект разрабатывается на **Godot 4.6.2 stable** с использованием GDScript.

## Обязательные документы разработки

Перед изменением кода или сцен необходимо прочитать:

- [Правила проекта](PROJECT_RULES.md)

Ключевые ограничения:

- запрещены монолитные классы, сцены и менеджеры;
- ни один исходный файл не может превышать 600 строк;
- примерно с 450 строк файл должен оцениваться на разделение;
- каждое изменяемое состояние имеет одного владельца;
- настраиваемые значения выносятся в ресурсы данных;
- UI не владеет игровой логикой;
- архитектурные изменения обновляют `ARCHITECTURE.md` в том же коммите.

## Основная идея

Платформа перемещается только по горизонтали и постоянно находится под воздействием ветра. Чтобы восстановить повреждённую секцию щита, игрок должен добраться до связанного с ней наземного энергетического шара, совместить его с шаром платформы и удерживать контакт.

Якоря позволяют стабилизировать платформу, но каждый установленный якорь одновременно становится путём, по которому враги могут подняться на борт.

## Текущая реализация

**Prototype 0.3** содержит:

- состояние забега и полную паузу;
- ветер направлений влево и вправо с силой 1–3;
- физическое рулевое усилие, инерцию и границы мира;
- тестовый энергетический контакт;
- четыре якоря и фиксированные точки крепления;
- отдельное управление, очередь установки и экстренное снятие;
- конечную длину каждого троса в обоих направлениях;
- направленное удержание и полную фиксацию;
- перегрузку одиночного натянутого якоря при ветре силы 3;
- срыв и автоматический возврат;
- декомпозированный AnchorDomain;
- data-driven `AnchorBalance`;
- регрессионный тест для правого якоря при ветре вправо;
- диагностический интерфейс без внешних ассетов.

Исправлена ошибка, при которой правый якорь мог бесконечно растягиваться вправо. Теперь при ветре силы 1–2 платформа останавливается на конечной длине, а при силе 3 начинается перегрузка и обрыв. Для левого якоря правило работает зеркально.

### Запуск

1. Установить Godot 4.6.2 stable.
2. Клонировать репозиторий.
3. Открыть `project.godot`.
4. Запустить проект клавишей `F5`.

### Управление

- `←` / `→` — рулевое усилие.
- `1–4` — установить или снять соответствующий якорь.
- `R` — снять все якоря.
- `Space` — нанести тестовой секции 10 урона.
- `Esc` — пауза.
