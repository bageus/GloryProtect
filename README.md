# GloryProtect

GloryProtect — survival roguelike с видом сбоку, в котором игрок управляет магически парящей платформой, поддерживает пять секций глобального щита и защищает экипаж от физического абордажа.

## Движок

Godot **4.6.2 stable**, строго типизированный GDScript.

## Обязательные документы

- [Правила проекта](PROJECT_RULES.md)
- [Архитектура проекта](ARCHITECTURE.md)
- [Инструкции для coding agents](AGENTS.md)

Основные ограничения:

- максимум 600 строк на поддерживаемый файл;
- каждое изменяемое состояние имеет одного владельца;
- баланс находится в типизированных ресурсах;
- UI не владеет игровой логикой.

## Текущая версия

**Prototype 1.7** содержит:

- горизонтальную платформу и ветер силы 1–3;
- четыре якоря с конечной длиной троса;
- пять наземных шаров и секций щита;
- три защитника, роли, замены и ближний бой;
- физический абордаж только через якоря;
- агрегированные стратегические волны и миникарту;
- экономику, статистику и итоговый экран;
- критическое визуальное и звуковое предупреждение;
- сетку размещаемых объектов по клеткам платформы;
- один медицинский пост и роль лекаря;
- несколько независимых турелей;
- отдельного оператора для каждой турели;
- автоматическую стрельбу по ближайшим физическим врагам;
- перенос и демонтаж турелей;
- две одинаковые карточки-заглушки без эффектов.

## Размещаемые объекты

`BuildableGrid` остаётся единственным владельцем занятости клеток и размещённых объектов.

Правила:

- одна клетка содержит не больше одного объекта;
- служебные клетки рулевого, якорщиков и двери замены недоступны;
- персонаж на клетке не мешает размещению;
- объекты не блокируют движение;
- открытие не теряется после демонтажа;
- медицинский пост существует максимум в одном экземпляре;
- турелей можно открыть до четырёх в текущем прототипе.

Координаты клеток вычисляет только `PlatformController`.

## Медицинский пост

Лекарь выбирает самого тяжело раненого живого защитника, при равенстве — ближайшего. После пяти секунд непрерывного лечения восстанавливается один сегмент. Начатый цикл завершается перед обычным переназначением.

При переносе поста текущий цикл продолжается. При демонтаже назначенный лекарь немедленно становится свободным бойцом.

## Турели

Компоненты:

```text
TurretRuntime
TurretTargetSelector
TurretSystem
TurretDebugInput
```

Каждая установленная турель:

- является отдельным динамическим постом роли `TURRET`;
- требует собственного защитника-оператора;
- без прибывшего оператора не работает;
- имеет круговой радиус действия;
- выбирает ближайшего живого врага;
- атакует врагов, поднимающихся по тросу, и врагов на платформе;
- наносит 1 урон за успешный выстрел;
- не перенаправляет уже начатый выстрел на новую цель;
- работает независимо от других турелей.

Несколько турелей могут одновременно выбрать одного врага. Если первый выстрел убивает цель, остальные уже начатые выстрелы проходят впустую.

Оператор турели не использует ближний бой. Если ему назначена новая роль во время выстрела, он сначала завершает текущий выстрел, затем уходит.

При переносе турель мгновенно появляется на новой клетке, а оператор физически перебегает к ней. Пока он не прибыл, турель не стреляет. Если перенос произошёл во время выстрела, выстрел завершается до начала перехода.

При демонтаже выстрел отменяется, пост роли удаляется, а оператор немедленно становится свободным бойцом. Открытие турели сохраняется.

```text
Кулдаун: 0.8 секунды
```

Параметры находятся в:

```text
resources/balance/buildable_balance.tres
```

## Карточки

Стоимость:

```text
5, 10, 15, …, 100, 200, 400, 800
```

Обе карточки пока одинаковы и не применяют улучшение. Медицинский пост и турели открываются только debug-командами.

## Управление прототипом

- `←` / `→` — рулевое усилие.
- `1–4` — установить или снять якорь.
- `R` — снять все якоря.
- `Esc` — пауза.
- `5`, `6`, `7` — выбрать защитника.
- `D` — рулевой.
- `Z` / `X` — левый / правый якорщик.
- `C` — свободный защитник.
- `H` — назначить лекарем.
- `,` / `.` — выбрать клетку платформы.
- `B` — открыть медицинский пост.
- `M` — установить или перенести медицинский пост.
- `Delete` — демонтировать медицинский пост.
- `T` — открыть ещё одну турель.
- `G` — установить новую турель.
- `K` — выбрать следующую установленную турель.
- `V` — перенести выбранную турель.
- `J` — назначить выбранного защитника оператором выбранной турели.
- `Backspace` — демонтировать выбранную турель.
- `F1–F5` — выбрать секцию щита.
- `Space` — тестовый урон секции.
- `Enter` — новый забег на итоговом экране.
