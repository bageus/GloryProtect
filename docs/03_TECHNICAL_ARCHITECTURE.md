# GloryProtect — техническая архитектура

Каноническая архитектура проекта находится в корневом файле:

- [`ARCHITECTURE.md`](../ARCHITECTURE.md)

Обязательные инженерные правила находятся в:

- [`PROJECT_RULES.md`](../PROJECT_RULES.md)

Этот файл сохранён только для совместимости со старой ссылкой в нумерованной документации. Архитектурные правила и границы систем здесь не дублируются, чтобы не создавать второй источник правды.

Любые архитектурные изменения должны вноситься непосредственно в `ARCHITECTURE.md` в том же коммите, что и соответствующие изменения кода.
