# GloryProtect — статус реализации

## Целевая версия движка

Проект разрабатывается на **Godot 4.6.2 stable** с использованием GDScript.

## Текущий этап

Реализован первый запускаемый каркас прототипа, соответствующий этапам 0–2 дорожной карты:

- создан `project.godot`;
- назначена главная сцена;
- добавлен контроллер состояния забега;
- добавлена стартовая безопасная задержка;
- добавлена полная ручная пауза;
- реализован ветер направлений влево и вправо;
- реализованы три уровня силы ветра;
- добавлены небольшие колебания фактической силы;
- реализовано физическое горизонтальное управление платформой;
- управление работает только при назначенном рулевом;
- добавлены инерция, сопротивление и ограничение скорости;
- добавлены невидимые границы тестового мира;
- камера следует за платформой;
- добавлен один тестовый наземный шар;
- добавлена зона энергетического контакта;
- добавлена тестовая секция щита;
- контакт восстанавливает тестовую секцию с постоянной скоростью;
- добавлен диагностический HUD;
- временная графика полностью рисуется кодом и не требует ассетов.

## Запуск

1. Установить Godot 4.6.2 stable.
2. Клонировать репозиторий.
3. Открыть файл `project.godot` через Project Manager.
4. Запустить проект клавишей `F6` или `F5`.

Главная сцена:

```text
res://scenes/game/game_root.tscn
```

## Управление прототипом

| Действие | Управление |
|---|---|
| Приложить силу влево | стрелка влево |
| Приложить силу вправо | стрелка вправо |
| Нанести тестовой секции 10 урона | Space |
| Включить или снять ручную паузу | Esc |

`Space` является только временным инструментом тестирования зарядки. В финальной игре урон секциям наносят стратегические враги.

## Архитектурные решения первого этапа

### GameFlowController

Хранит единое состояние забега:

```text
BOOT
START_DELAY
RUNNING
CARD_SELECTION
MANUAL_PAUSE
GAME_OVER
```

Работает в режиме `PROCESS_MODE_ALWAYS`, чтобы снимать паузу через `Esc`, когда дерево сцены остановлено.

### WindSystem

Не изменяет платформу напрямую. Предоставляет текущее горизонтальное усилие через `get_current_force()`.

### PlatformController

Получает:

- доступность рулевого;
- ввод игрока;
- силу ветра.

Хранит горизонтальную скорость и применяет ограничения мира.

### PrototypeWorld

Пока представляет только тестовый уровень, один шар и визуализацию зоны контакта. В дальнейшем будет заменён системой пяти наземных шаров и фиксированных точек якорей.

### PrototypeShieldSystem

Временная реализация одной секции для проверки контакта. Не является финальной архитектурой пяти секций.

## Проверка

GDScript-файлы проверены статическим линтером `gdlint 4.5.0`: синтаксических проблем не обнаружено.

Полноценный импорт через редактор Godot необходимо также проверять после клонирования, поскольку среда разработки репозитория не предоставляет встроенный исполняемый файл Godot.
