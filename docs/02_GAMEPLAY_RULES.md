# GloryProtect — справочник игровых правил

Этот файл является навигационным индексом. Подробные правила разделены по доменам, чтобы каждый документ оставался меньше 600 строк и имел одну ответственность.

## Источники правды

1. [Забег, мир, шары и щит](rules/01_CORE_WORLD_AND_SHIELD.md)
2. [Платформа, ветер и якоря](rules/02_PLATFORM_WIND_AND_ANCHORS.md)
3. [Стратегические волны и физический абордаж](rules/03_STRATEGIC_AND_BOARDING_THREATS.md)
4. [Экипаж, роли и ближний бой](rules/04_CREW_COMBAT_AND_ROLES.md)
5. [Объекты, экономика и интерфейс](rules/05_BUILDABLES_ECONOMY_AND_UI.md)
6. [Карточки улучшений, ветки и специализации](rules/06_UPGRADE_CARDS_BRANCHES_AND_SPECIALIZATIONS.md)

## Правило обновления

- Детальное правило должно находиться только в одном тематическом файле.
- GDD может кратко описывать механику, но не должен дублировать полную спецификацию.
- При изменении механики обновляется соответствующий тематический документ.
- Новая механика добавляется в существующий домен либо получает отдельный файл, если текущий приблизился к 450 строкам.
